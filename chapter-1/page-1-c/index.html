JRE Emulation

JRE Emulation

Bytecoder comes with a JRE emulation layer which is based on OpenJDK and also a set of additional libraries for browser interaction, vue.js and OpenCL integration.

Bytecoder comes with a set of JPMS modules like java.base etc. Please note that Bytecoder does not support the JPMS in general. From the compiler and runtime view, all classes are exported to the ALL-UNNAMED module by default, module-info declarations are completely ignored.

bytecoder-core

Bytecoder-core includes the compiler Logic and the JUnit Testrunner. Most of the time you won’t need this module directly, as it is included in the CLI and the Maven Plugin by default.

<dependencies>
    <dependency>
        <groupId>de.mirkosertic.bytecoder</groupId>
        <artifactId>bytecoder-core</artifactId>
        <version>2023-05-19</version>
    </dependency>
</dependencies>
